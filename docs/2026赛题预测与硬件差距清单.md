# 2026 TI 杯赛题预测与硬件差距清单（控制/视觉方向）

输入三源：① 2026 官方仪器/物料清单（用户提供，2026-07-21）；② 历届题目谱系（2024 TI 杯
与 2025 国赛已联网核实，见文末来源）；③ 本仓库已建成能力（chassis/line_follow/motion/route
+ gimbal 视觉三闭环）。预测逻辑 = 题目惯性：TI 杯（偶数年）惯例做「上届国赛热点的减法变体」
+ 自身偶数年谱系延续。

## 1. 物料清单信号解码（先把噪声滤掉）

清单里一大半与控制/视觉无关，是其他题型的固定配料，不必恐慌：

| 清单项 | 归属 | 判定 |
|---|---|---|
| 磁环/铁芯磁环/镀锌铁片/漆包线(部分) | 电源类（自绕变压器/电感） | 忽略 |
| 自耦调压器/隔离变压器/三相可变电阻/滑线变阻器/功率分析仪 | 电源类 | 忽略 |
| DDS/AD-DA/运放/比较器/乘法器/宽带放大器 | 信号源/放大器/仪表类 | 忽略 |
| FPGA/DSP/嵌入式板/最小系统板 | 通用平台项，年年都在 | 忽略 |
| 四旋翼(≤450mm)+图传 | 飞行器题（几乎必有一道） | 不在备赛范围，放弃 |

**与控制/视觉强相关的信号**（按强度排）：

| 信号 | 解读 |
|---|---|
| **TI MSPM0 系统板（可含板载调试）** | 2024 H 先例：指定 MSPM0 的控制题**再来一道**几乎板上钉钉（我们的 G3519 直接合规） |
| **电动小车 ≤35×25cm，禁履带+麦轮** | 小车题必出。2025 国赛本科组没有小车题 → 憋了一届，2026 出的概率更高。禁麦轮=差速转向（正是现底盘）。**尺寸从 2024 的 25×15 放宽到 35×25 → 暗示车上要装东西**（摄像头/云台/机构） |
| **摄像头（可带处理器模块）** | 视觉题/视觉小车。「可带处理器」= 允许 OpenMV/K210/树莓派类上位机——正是现架构（视觉上位机出坐标，MCU 伺服） |
| **短距离无线通信模块** | 双车协同或车-装置协同（2021 F 双车血统），更可能作为发挥部分 |
| **无线图像传输模块** | 视觉遥操作/远程监控（2025 H 野生动物巡查血统），或与四旋翼配对 |
| **小型电磁铁+钢球(1cm)+漆包线** | 双解：a) 2024 F 磁悬浮直系复现（偏模拟，非我们方向）；b) 小车/云台的**吸取-搬运-投放机构**（配电子秤 300~500g 称载荷） |
| **角度传感器+多功能量角器** | 云台/摆角度验收（2025 E 瞄准装置的场地验收就靠量角） |
| **4 分 PPR 水管(20mm)** | 双解：场地标志杆/门框/挡板（低冲击），或摆杆（倒立摆/风力摆回归，低概率高冲击） |
| **秒表/卷尺/光电传感器/蜂鸣器/LED/数码管/继电器** | 小车计时定距 + **声光提示**（2024 H 每个检查点响一次，历届标配）——固件 Q9 缺口 |

## 2. 历届控制/视觉题谱系（已核实）

| 年 | 赛别 | 控制/视觉题 | 与本仓库的血缘 |
|---|---|---|---|
| 2021 | 国赛 | F 智能送药小车（循迹+数字识别+双车接力） | 底盘链 + 无线协同 |
| 2023 | 国赛 | E 运动目标控制与自动追踪系统（双云台激光） | gimbal 直系（旧 2DPlatform_LaserStrike 就是为它备的） |
| 2024 | **TI杯** | **H 自动行驶小车**（指定 MSPM0G3507，禁履带麦轮，A→B 无标线+半弧线，声光提示，4 圈计时，25×15cm）；E 三子棋（视觉+机械臂）；F 磁悬浮（电磁铁+钢球）；D 盘点无人机 | H=底盘链直系；F=清单电磁铁钢球的另一去处 |
| 2025 | 国赛 | **C 基于单目视觉的目标物测量装置**；**E 简易自行瞄准装置**；H 野生动物巡查系统。**本科组无小车题** | C/E = 摄像头+云台直系；小车空窗一届 |

惯性规律：TI 杯控制题连续两届（2022 送货/跟随、2024 自动行驶）都是**指定 TI MCU 的小车题**；
国赛刚考过的热点（2025 C 单目测量 / E 自行瞄准）在下一届 TI 杯以减法变体下放是常态。

## 3. 预测（按概率排）

### Tier 1 —— 主攻（与备赛直接对口）

1. **视觉自动行驶小车（概率最高）**：2024 H 直系 + 2025 视觉热点下放。形态：循迹 +
   无标线区航位推算 + 弧线/定点停车 + 沿途视觉识别（数字/图形/目标物）+ 声光提示 + 计时。
   依据：MSPM0 板再次点名、车尺寸放宽（装摄像头）、秒表/卷尺/光电/蜂鸣器全套在列。
   仓库覆盖 ≈85%：chassis/line_follow(12 路灰度+元素检测)/motion(直行/转/弧)/route 段表
   /odometry 全就绪；缺声光（Q9）与赛题段表（T01，设计上就是留给赛题的）。
2. **云台视觉瞄准/追踪（独立题或车载复合，概率次高）**：2025 E + 2023 E 直系。形态：
   静/动目标识别 → 云台指向（激光或摄像头对准）→ 角度/命中验收（量角器+角度传感器在列）。
   仓库覆盖 ≈80%：gimbal 三闭环 + GimbalTune 调参链完备；缺动目标前馈（odometry 读点已留）。
3. **车 + 云台复合（概率中高，最值得押）**：车行驶中识别目标并瞄准/投放。清单同时给了
   车+摄像头+图传+电磁铁+钢球+电子秤——「小车沿线行驶，识别目标点，电磁铁吸/放钢球或
   激光指示」这类复合题能同时解释这串物料。恰好是仓库两条链的合体，只缺 T01 编排 + 机构件。

### Tier 2 —— 保险（备料不投入主力）

4. **磁悬浮/电磁装置**（2024 F 直系，电磁铁+钢球+漆包线复现）——偏模拟电路，抽中即重开，不备。
5. **双车协同**（无线模块信号）——更可能是 Tier1 题的发挥部分；全双车题需第二套底盘，成本大，
   建议只预研无线 Driver 不买整车。
6. **摆类回归**（PPR 管+角度传感器）——低概率高冲击；PPR 管按场地材料理解更合理。

## 4. 硬件清单（做测试/采购，按优先级）

**已有已验证**：G3519 板、TB6612 差速底盘+编码器、12 路灰度(NCHD1)、IMU 模组、
EMM42 双轴步进云台、视觉上位机（摄像头带处理器）、OLED/按键、VOFA 链路、参数 flash。

**A 必备（Tier1 直接要用，立即办）**：

| # | 项 | 用途/动作 |
|---|---|---|
| A1 | 蜂鸣器 + LED（+限流/驱动） | 声光提示=历届小车题硬指标（Q9 缺口）。定引脚→补小 Driver→hmi 或独立包装 |
| A2 | 车体尺寸复核 | 量现车长宽 ≤35×25cm？（2024 只给 25×15，2026 放宽——确认合规并留装摄像头空间） |
| A3 | 车载摄像头/云台供电与安装 | 云台上车一体化：供电电流、重心、抖动实测（车动瞄准是复合题核心） |
| A4 | 场地材料 | 白底+黑线胶带、A/B/C/D 点标志、卷尺/秒表/量角器（清单同款，自建 2024 H 式场地演练） |
| A5 | 小型电磁铁 + MOS/继电器驱动 + 钢球 1cm | 吸-放机构预研（复合题机构件；GPIO 一路即可，先台架验吸放力与功耗） |
| A6 | 电池续航/稳压复测 | 4 圈连跑计时题：全程电压跌落对速度环/灰度阈值的影响 |

**B 保险（低成本先备，不动主力）**：

| # | 项 | 用途/动作 |
|---|---|---|
| B1 | 短距离无线模块（UART 型，2.4G/BLE 透传） | 双车/协同发挥部分。注意串口资源：UART0 现为 BSL_ENTRY——先做口规划再买 |
| B2 | 角度传感器（磁编/电位器式） | 云台角度验收备份 + 摆类保险 |
| B3 | PPR 水管数根 + 底座 | 场地标志杆/门框自建 |
| B4 | 光电对管 | 测速/计圈备份 |
| B5 | 电子秤(300~500g) | 载荷称量（借用即可，不必买） |

**C 不投**：四旋翼整机、磁悬浮线圈料、第二套整车（除非赛前确认双车题）。

## 5. 固件预制排序（对应预测，赛前可做）

1. **Q9 声光 Driver**（小时级）：蜂鸣器/LED 引脚定案 → driver 小模块 + Service 包装；
2. **T01 段表实弹演习**：按 2024 H 场地（A→B 直行→半弧→计时 4 圈）写一份 route 段表跑通——
   route/motion 的设计目标就是「新题=换段表」，赛前必须演习一次全流程；
3. **gimbal 运动前馈接线**（契约已留 odometry 读点）：车动瞄准的稳态滞后对策，复合题保险；
4. **无线 Driver**（仅当决定备双车）：透传 UART 收发 + 心跳超时停车（§8.1 口径）。
