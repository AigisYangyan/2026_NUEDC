# 2026 TI 杯赛题预测与硬件差距清单（控制/视觉方向）

输入三源：① 2026 官方仪器/物料清单（用户提供，2026-07-21）；② 历届题目谱系（2024 TI 杯
与 2025 国赛已联网核实，见文末来源）；③ 本仓库已建成能力（chassis/line_follow/motion/route
+ gimbal 视觉三闭环）。预测逻辑 = 题目惯性：TI 杯（偶数年）惯例做「上届国赛热点的减法变体」
+ 自身偶数年谱系延续。

## 1. 物料清单信号解码（先把噪声滤掉）

清单里一大半与控制/视觉无关，是其他题型的固定配料，不必恐慌：

| 清单项 | 归属 | 判定 |
|---|---|---|
| 磁环/铁芯磁环/镀锌铁片/漆包线(部分) | 电源类（自绕变压器/电感）；**另一解读**：无线充电线圈（2019 A 动态无线充电小车 / 2022-10 A 无线充电循迹小车用的就是磁环+漆包线耦合） | 主判电源噪声，保留无线充电小车次解读（§3 Tier2） |
| 自耦调压器/隔离变压器/三相可变电阻/滑线变阻器/功率分析仪 | 电源类 | 忽略 |
| DDS/AD-DA/运放/比较器/乘法器/宽带放大器 | 信号源/放大器/仪表类 | 忽略 |
| FPGA/DSP/嵌入式板/最小系统板 | 通用平台项，年年都在 | 忽略 |
| 四旋翼(≤450mm)+图传 | 飞行器题（几乎必有一道） | 不在备赛范围，放弃 |

**与控制/视觉强相关的信号**（按强度排）：

| 信号 | 解读 |
|---|---|
| **TI MSPM0 系统板（可含板载调试）** | 2024 H 先例：指定 MSPM0 的控制题**再来一道**几乎板上钉钉（我们的 G3519 直接合规） |
| **电动小车 ≤35×25cm，禁履带+麦轮** | 小车题必出。2025 国赛本科组没有小车题 → 憋了一届，2026 出的概率更高。禁麦轮=差速转向（正是现底盘）。**尺寸从 2024 的 25×15 放宽到 35×25 → 暗示车上要装东西**（摄像头/云台/机构） |
| **摄像头（可带处理器模块）** | 视觉题/视觉小车。「可带处理器」= 允许 OpenMV/K210/树莓派类上位机——正是现架构（视觉上位机出坐标，MCU 伺服） |
| **短距离无线通信模块** | 双车协同或车-装置协同（2021 F 双车血统），更可能作为发挥部分 |
| **无线图像传输模块** | 视觉遥操作/远程监控（2025 H 野生动物巡查血统），或与四旋翼配对 |
| **小型电磁铁+钢球(1cm)+漆包线** | 三解：a) 2024 F 磁悬浮直系复现（偏模拟，非我们方向）；b) 小车/云台的**吸取-搬运-投放机构**（配电子秤 300~500g 称载荷）；c) 配水管=2017 M 管道钢珠测量血统（仪表类） |
| **电子秤(300~500g)** | **送药/送货载荷称量直系信号**——2021 F 智能送药小车的验收就用电子秤称药品载荷；量程 300~500g 恰是「小车载物」量级，不是电源题用具 |
| **角度传感器+多功能量角器** | 云台/摆角度验收（2025 E 瞄准装置的场地验收就靠量角） |
| **4 分 PPR 水管(20mm)** | 双解：场地标志杆/门框/挡板（低冲击），或摆杆（倒立摆/风力摆回归，低概率高冲击） |
| **秒表/卷尺/光电传感器/蜂鸣器/LED/数码管/继电器** | 小车计时定距 + **声光提示**（2024 H 每个检查点响一次，历届标配）——固件 Q9 缺口 |

## 2. 历届控制/视觉题谱系（2013–2025 全表已核实；近两届为主要依据）

|---|---|---|
| 2017 | L 自动泊车系统 | 循迹+泊车动作 |
| 2019 | A 电动小车动态无线充电 | 循迹 + 磁环线圈无线供电 |
| 2019 | B 巡线机器人 | 线网巡检、岔路决策 |
| 2021 | **F 智能送药小车** | **循迹 + 岔路 + 摄像头识别病房号(数字) + 载荷(电子秤称药) + 装卸 LED 指示 + 双车无线接力** |

| 2019 | H/J 模拟电磁曲射炮 | 电磁铁+弹丸 |
| 2023 | I 气垫悬浮车 | 悬浮 |
| 2024 | F 磁悬浮实验装置 | 漆包线+电磁铁+钢球 |

惯性规律（三条）：① TI 杯控制题连续两届是**指定 TI MCU 的小车题**（2022 跟随、2024 自动行驶），
2025 国赛小车空窗一届 → 2026 小车题概率进一步抬升；② 国赛刚考的热点在下届 TI 杯做减法下放
（2025 C/E 视觉+瞄准 → 2026 视觉元素大概率进题）；③ 经典小车题每隔几年**魔改复出**
（泊车 2017L→2022B、无线充电 2019A→2022A、跟随 2022C/H），**送药小车(2021F)按此节奏正排在
复出窗口**——且 2026 清单把它的全套验收配料又列了一遍（电子秤/无线模块/摄像头/LED）。

## 3. 预测（按概率排）

### Tier 1 —— 主攻（与备赛直接对口）

1. **经典小车魔改——送药/送货型（概率最高，物证最全）**：2021 F 智能送药小车按「泊车
   2017L→2022B、无线充电 2019A→2022A」的复出节奏正排在魔改窗口，且 2026 清单把它的
   **全套验收配料又列了一遍**：电子秤(300~500g 称载荷)、短距离无线模块(双车接力)、
   摄像头可带处理器(病房号/标识识别)、LED(装卸指示)、光电/秒表/卷尺(循迹计时)。
   预期形态：循迹网格 + 岔路决策 + 视觉识别标识(数字/颜色/图形) + 定点装卸载荷 +
   声光提示 +（发挥部分）双车无线接力。
   仓库覆盖 ≈75%：line_follow 的**元素检测(左岔/右岔/横线)+route FOLLOW_UNTIL 就是为
   岔路网格设计的**；缺三件——视觉组的标识识别（需预训练）、无线 Driver（Q8）、载荷装卸机构。
2. **2024 H 式自动行驶魔改（概率并列高）**：无标线区航位推算 + 弧线/定点停车 + 计时圈，
   可能叠加 2025 视觉热点（沿途识别目标物）。MSPM0 板点名 + 车尺寸放宽是直接证据。
   仓库覆盖 ≈85%：motion 剖面直行/定角转/圆弧 + odometry + route 全就绪；缺声光（Q9）。
3. **云台视觉瞄准/追踪（独立题或车载复合）**：2025 E + 2023 E 直系，一届三视觉题的热度
   顺延。形态：静/动目标识别 → 云台指向（激光/摄像头对准）→ 角度/命中验收（量角器+角度
   传感器在列）。仓库覆盖 ≈80%：gimbal 三闭环 + GimbalTune 调参链完备；缺动目标前馈
   （odometry 读点已留）。**车+云台复合**（行驶中瞄准/吸放钢球）能同时解释
   车+摄像头+电磁铁+钢球+电子秤这串物料，是两条链的合体，最值得押注的复合形态。

### Tier 2 —— 保险（备料不投入主力）

4. **无线充电循迹小车**（2019 A→2022-10 A 血统；磁环+漆包线的次解读）——底盘链与 Tier1
   完全同款，差异全在无线电能收发（硬件组工作）；固件侧零额外投入即天然覆盖。
5. **磁悬浮/电磁装置**（2024 F 直系，电磁铁+钢球+漆包线复现）——偏模拟电路，抽中即重开，不备。
6. **双车协同全题**（2022-07 C/H 跟随直系）——更可能以 Tier1-1 发挥部分出现；全双车题需
   第二套底盘，成本大，只预研无线 Driver 不买整车。
7. **球/摆/管类**（2017 B 滚球、2017 M 管道钢珠、2013 C 倒立摆血统；PPR 管+钢球+角度传感器
   可拼出任一）——低概率高冲击；PPR 管按场地标志材料理解更合理，不预建。

## 4. 硬件清单（做测试/采购，按优先级）

**已有已验证**：G3519 板、TB6612 差速底盘+编码器、12 路灰度(NCHD1)、IMU 模组、
EMM42 双轴步进云台、视觉上位机（摄像头带处理器）、OLED/按键、VOFA 链路、参数 flash。

**A 必备（Tier1 直接要用，立即办）**：

| # | 项 | 用途/动作 |
|---|---|---|
| A1 | 蜂鸣器 + LED（+限流/驱动） | 声光提示=历届小车题硬指标（Q9 缺口）。定引脚→补小 Driver→hmi 或独立包装 |
| A2 | 车体尺寸复核 | 量现车长宽 ≤35×25cm？（2024 只给 25×15，2026 放宽——确认合规并留装摄像头空间） |
| A3 | 车载摄像头/云台供电与安装 | 云台上车一体化：供电电流、重心、抖动实测（车动瞄准是复合题核心） |
| A4 | 场地材料 | 白底+黑线胶带、A/B/C/D 点标志、卷尺/秒表/量角器（清单同款，自建 2024 H 式场地演练） |
| A5 | 小型电磁铁 + MOS/继电器驱动 + 钢球 1cm | 吸-放机构预研（复合题机构件；GPIO 一路即可，先台架验吸放力与功耗） |
| A6 | 电池续航/稳压复测 | 4 圈连跑计时题：全程电压跌落对速度环/灰度阈值的影响 |
| A7 | 短距离无线模块（UART 透传型，买一对） | **随送药魔改判断升为 A 级**：双车接力是 2021 F 发挥分主体。注意串口资源——UART0 现为 BSL_ENTRY，先定口再接线 |
| A8 | 视觉组：标识识别预训练 | 送药魔改的识别对象大概率是**数字/颜色/图形标识**（2021 F 是红色病房号）——视觉上位机提前建训练集与识别管线，坐标协议已就绪可直接复用 |
| A9 | 载荷装卸机构 + 装/卸指示灯位 | 2021 F 式：载荷放置检测（微动开关/光电）+ 红绿 LED 指示，与 A1 声光合并设计 |

**B 保险（低成本先备，不动主力）**：

| # | 项 | 用途/动作 |
|---|---|---|
| B1 | 角度传感器（磁编/电位器式） | 云台角度验收备份 + 摆类保险 |
| B2 | PPR 水管数根 + 底座 | 场地标志杆/门框自建 |
| B3 | 光电对管 | 测速/计圈备份 |
| B4 | 电子秤(300~500g) | 载荷称量（借用即可，不必买） |

**C 不投**：四旋翼整机、磁悬浮线圈料、第二套整车（除非赛前确认双车题）。

## 5. 固件预制排序（对应预测，赛前可做）

1. **Q9 声光 Driver**（小时级）：蜂鸣器/LED 引脚定案 → driver 小模块 + Service 包装——
   Tier1 三个场景全都要它；
2. **T01 段表实弹演习，两套场地**：a) 2024 H 场地（A→B 直行→半弧→计时 4 圈）；
   b) **2021 F 式岔路网格**（FOLLOW_UNTIL 左岔/右岔/横线 + 标识触发定点停车装卸）——
   route/motion/track_elements 的设计目标就是「新题=换段表」，赛前各演一遍全流程；
3. **无线 Driver**（随送药魔改判断提前）：UART 透传收发 + 帧校验 + **心跳超时停车**
   （§8.1 命令过期即停口径）；双车接力协议等题目出来再定，Driver 先就位；
4. **gimbal 运动前馈接线**（契约已留 odometry 读点）：车动瞄准的稳态滞后对策，复合题保险。
