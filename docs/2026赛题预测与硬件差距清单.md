## 1.5 清单文字游戏解码（R1~R11，回测依据见数据层 §2/§4）

**R1 独特簇规则**：一道题的独门道具总是**成簇**出现——2025「UV感光纸+405nm激光+
二维云台+量角器+车规」→ E 瞄准；2024「柱状铁芯+霍尔」→ F 磁悬浮、「超声换能器+
配套变压器」→ G 录音屏蔽；2023「红绿激光笔+双云台」→ E 追踪。成簇低频道具归题
回测命中 ≈97%。

**R2 孤立单件不承重**：清单是多组别共享超集，约 10% 低频件最终无归属（确证负例：
2024 孤立激光笔；未归属候选：2019 三线激光/三脚架、2021 空心透明球、2021/2023
亚克力板+钕铁硼磁铁）。**只对簇下注，永不对单件下注。**

**R3 精度即道具**：「约」字+反常精度=官方指定采购件、验收尺寸依赖它。2026「4分PPR
水管（外径约20mm，壁厚约3.4mm）」→ 内径 20−2×3.4=**13.2mm**；「钢球（直径约1cm）」
→ 管内游隙 3.2mm——一对算过的配合尺寸。裸名词（漆包线、蜂鸣器）=通用耗材，无信息量。

**R4 规则前置=题目已成稿**：2026 清单车条目「不得采用履带和麦氏轮」是 2024H/2025E
题面原句——约束从已写好的题目文本抄进清单。

**R5 仪器定界**：2026 对比 2025 国赛：示波器 500→100MHz、信号源 100→20MHz、万用表
6位半→5位半，删频谱仪/射频源/LCR/电能质量分析仪 → 无高频题、无接收机、无精密仪表
题；功率分析仪+调压/隔离变压器保留 → 电源题在。仪器上限≈题目测量能力上限。

**R6 增删差分（2026 vs 2025 国赛清单）**：

| 方向 | 条目 | 解读 |
|---|---|---|
| **删** | **二维电动云台、红色激光笔、405nm 蓝紫激光笔、UV 感光纸** | 瞄准题独门簇**整簇消失** → 激光瞄准/打靶形态出清（R11） |
| 删 | 舵机、麦克风、扬声器 | 机械臂/弈棋类、声学类出清 |
| 删 | 网线/RJ45、变容二极管、高频磁环/高频电感、超级电容 | 网线仪/接收机不复现；无线充电车降权 |
| 增 | 短距离无线模块、无线图传、电子秤(300~500g) | 送货式车簇补全（协同+载荷） |
| 增 | 小型电磁铁、钢球(1cm)、PPR 水管(内径13.2) | 球-管配合簇（2017M 直系） |
| 增 | 铁芯磁环(工频大环)、镀锌薄铁片(0.3mm=变压器叠片) | 工频电源题，噪声确认 |
| 变 | 小车 25×15(限高25,轮3~4) → **35×25(无限高无轮数)** | 车放大 2.3 倍+车顶放开=装载形态（R8） |

**R7 车规定律（满样本）**：清单车条目=当年某题（或其载体）的题面车规原文，**10/10**，
一手逐字核实 8 例——2017C 信标车 20×15、2018C 30×26、2019A 30×26、2020C 25×25、
2021F 25×20×25、2023G 消防车 25×35×40、2024H 25×15×15、2025E 25×15×25。
泊车题用**反向下限**车规（2017L 车≥26×16、2022-10B 车≥25×15）——车位固定、车不许
过小，同属此定律。⇒ **2026 的 35×25 禁麦轮车题已成稿**，题面将自带高度上限
（2023G 先例：清单不写高、题面给 40cm 量级）。

**R8 限高是形态分水岭**：2024 限高 15cm+题面**明令禁装摄像头**=纯行驶题；2025 限高
25cm=车载云台复合；2026 **无限高+footprint 放大**=车顶装载结构（载荷仓/摄像头桅杆）
形态信号成立，且 2024 的摄像头禁令语义反转（清单主动给摄像头）。

**R9 验收工具直译**：仪器组小工具是验收条款的倒影——秒表=计时分；5米卷尺=米级场地；
1米直尺=厘米形位；多功能量角器=角度验收（2021D「θ 角的标定可利用量角器」题面原句；
2017M 倾角显示误差≤3°）；**电子秤 300~500g=载荷质量验收**（2021F 药品约 200g 是题面
参数、当年清单无秤；2026 秤入仪器组=质量指标硬化为当场称量）。

**R10 图传+大屏簇**：无线图传+显示屏≥6吋+摄像头=远程实时图像监控——去向依次：
四旋翼 FPV（最常见）、空地协同的车载显示（2023G 消防车显示器直系）、车题发挥部分
（识别结果回传）。

**R11 机电道具必上清单，视觉耗材不上清单**（双向边界，一手实证）：
- **机电执行器/专用器材**必上清单：2025E 连感光纸都列（云台+双激光+感光纸四件全）；
  2024 列舵机（E 三子棋机械臂）；2023 列双色激光+云台。**缺席即强反证**：2026 无云台/
  无激光/无感光纸/无舵机 → 一切瞄准/机械臂形态趋零。「小型电机及驱动器可自建云台」
  的兜底不成立——2025 清单同样有小电机条目仍单列云台。
- **视觉识别对象/被测物**从不上清单：2021F 数字纸张、2023G 自制火源 LED、2025C 被测
  物、2025H 识别对象均不在清单。视觉题的清单签名=摄像头+（可选）图传/大屏+验收量具。
  **摄像头道具缺席不构成对识别/测量题的反证。**
- 支架细节：2021 清单三脚架=2021D 摄像节点/悬摆支架（一手）；2026 无三脚架 →
  摄像头装在车上/装置上，与簇 1 互证。

**2026 三大道具簇**（R1 证据链）：
- **簇 1（最完整）**：小车(35×25,禁麦轮,无限高) + 摄像头(可带处理器) + 短距离无线 +
  电子秤(仪器组=载荷验收) + LED/蜂鸣器 + 光电 + 秒表/卷尺 → **送货/送药式视觉车**。
  载荷=普通物件（300~500g 量程对得上；钢球仅 4g 不属此簇）；基础线装卸按 2021F 原样
  大概率人工放取（TI 杯做减法，勿过度设计）；
- **簇 2（配合尺寸实锤）**：PPR管(内径13.2) + 钢球(10mm，与管内径滚动配合) +
  电磁铁(隔 3.4mm 非磁性管壁吸引/制动球) + 角度传感器(管倾角) + 量角器/直尺 →
  **管道钢球位置控制**。直系祖先 2017M 用的就是「市售非透明 4 分（外径约 20mm）硬塑料
  管材+钢珠+倾角显示」（题面原句同源），2026 加电磁铁=从测量题升级为控制题；
- **簇 3（噪声确认）**：铁芯磁环 + 镀锌铁片 + 漆包线 + 调压器/隔离变压器 + 功率分析仪
  → 工频电源题，与控制/视觉无关。

## 3. 预测（按概率排；权重与数据层模型一致）

### 3.5 加权预测

**结构判断**：R7 确定车题成稿，只对「车题形态」和「并行装置题槽位」分别加权。
难度定标见下方「平台第 3 年」条。

**车题形态分布（P(车题)≈1）**：

| 形态 | 权重 | 主证据 | 反证/风险 |
|---|---|---|---|
| **送药/送货式：识别+岔路+定点装卸(+双车/图传发挥)** | **0.45** | 簇 1 全套；2021F 五年复出窗口；电子秤入仪器组（R9）；无限高+大车=装载落定（R8）；识别对象不上清单（R11）；无三脚架=摄像头在车上 | 识别依赖视觉组产能 |
| 视觉增强自动行驶（2024H+摄像头解禁：航位推算+弧线+沿途识别） | 0.25 | R4 措辞抄题；MSPM0 点名惯例；秒表/卷尺（R9）；2024 禁摄像头→2026 给摄像头的语义反转（R8） | 纯行驶解释不了删限高+车放大 |
| **空地协同变体（车为地面执行端）** | 0.15 | 车规与 2023G 逐字相同；图传+大屏+四旋翼+短距无线复现 2023G 簇 5/6；3 年复出窗口 | 需无人机组产能；对纯控制组是半题 |
| 双车跟随全题（2022C/H 复刻） | 0.10 | 短距离无线在 | 无测距独门道具（无超声/UWB） |
| 无线充电循迹（2019A/2022-10A 复刻） | 0.05 | 磁环+漆包线次解读 | **无超级电容**（2018/2019 簇储能件缺席）；磁环规格=工频大环 |
| 车载云台瞄准 | ≈0 | 无 | R11 出清：云台/激光/感光纸全缺席 |

**平台第 3 年难度定标**（数据层 L6）：指定平台第 1 年=地板难度，逐年加一层复合，
第 3 年出协同/复合、省联赛难度逼近国赛——上代实证 2018 单功能→2020 单车闭环→
**2022 双车协同**；本代 2024 纯行驶(禁摄像头)→2025 车载云台复合→**2026 预期
基础线即含识别或机构一层复合，双车/协同占发挥分主体**（2021F 先例：发挥 44/50 全是
双车）。「省赛<国赛」不作为约束。

> **复杂度校准**：一切形态按「基础一条主线+发挥一两个加码」估计——2024H 基础是
> "跑圈+响一声"，2024F 是"悬住"。预测想定按上限写，**备赛与验收按基础线优先**：
> 送药式基础线大概率=人工装载+识牌+循迹到点+停车声光（自动装卸/双车全是发挥分）；
> 球管题基础线=把球停到指定刻度保持。

**押注结论（对本仓库两条链）**：
- **底盘链被主攻题用上的概率 ≈0.95**——前三形态合计 0.85 全部踩在
  chassis/line_follow/motion/route 上；**视觉的正确形态是「识别」不是「瞄准」**；
- **gimbal 链 ≈0——R11 出清**：机电道具必上清单而 2026 全缺席。gimbal 封存为已建成
  资产：零维护、零追加投入、不进任何备赛动作清单；
- **视觉备赛重心**：识别管线（数字/图形/颜色预训练+识别结果帧协议 P2），不是云台伺服；
- 缺口投入排序：声光(全形态要) > 标识识别预训练+P2 协议(0.45 头名+0.30 装置槽双押) >
  无线 Driver(0.45+0.15+0.10 三押) > **球管套件+电磁铁**(<50 元，装置槽头名 0.50 对冲)
  > 载荷在位检测+指示灯（基础线人工装卸，自动装卸降为发挥项）。

### Tier 2 —— 保险（备料不投入主力）

- **管道钢球位置控制（装置槽头名，0.50）**：若出，PID/param_tune/VOFA 调参框架整体
  迁移，机构与被控对象全新（≈3 天重开量级）。**便宜对策**：水管+钢球+电磁铁 <50 元
  先买齐；抽中日路径=pid(middleware 现成)+新 driver(电磁铁 PWM/倾角检测)。参照 2017M
  题面：管长 500mm、倾角 10~80°、钢珠计数/方向/倾角显示——2026 控制版预期在此之上
  加「定点停球+保持」；
- **双车协同**：更可能以送药发挥分出现；只预研无线 Driver 不买第二套整车；
- **无线充电循迹**：底盘链天然覆盖，无线电能收发归硬件组，固件零额外投入；
- **磁悬浮/电磁装置**：偏模拟电路，抽中即重开，不备。

## 5. 固件预制排序（对应预测，赛前可做）

1. **Q9 声光 Driver**（小时级）：蜂鸣器/LED 引脚定案 → driver 小模块 + Service 包装——

2. **T01 段表实弹演习，两套场地**：a) 2024H 场地（直行→半弧→计时 4 圈）；

3. **无线 Driver**：UART 透传收发 + 帧校验 + **心跳超时停车**（§8.1 命令过期即停口径）；
   双车协议等题目出来再定，Driver 先就位；
4. **簇 2 抽中日预案**：电磁铁 PWM Driver + 倾角读取 + pid 迁移路径（B2 套件台架数据
   先行）。
