# 全国大学生电子设计竞赛（国赛）本科组·控制类赛题原文档

已明确排除：无人机/飞行器类题目、高职高专组题目。

所有 PDF 均已验证：非 0 字节、文件头为 `%PDF`，且与 GitHub 存档（chenshuo/nuedc）逐字节 MD5 一致（交叉核对通过）。
