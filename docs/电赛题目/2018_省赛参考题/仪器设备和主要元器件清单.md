# 2018年TI杯联赛仪器仪表及元器件清单

> 来源: https://nuedc.xidian.edu.cn/html/news/2018/0711/387.html
> 发布时间: 2018-07-12
> 说明: 陕西赛区转发的TI公司说明，8道题（6道本科+2道高职）

**仪器仪表**: 双踪示波器、任意波形发生器、直流稳压源（恒流恒压模式自动切换）、五位半数字万用表、调频收音机等

**主要元器件**:
- FDC2214
- MSP430F5529
- TPS63020
- 带防撞圈的多旋翼飞行器，最大轴间距50厘米
- 小型电动车，外形尺寸不大于30厘米×26厘米
- 超级电容（法拉电容）
