# 2020年TI杯大学生电子设计竞赛 非接触物体尺寸形态测量（G题）

> 来源: https://www.nuedc-training.com.cn/index/news/details/new_id/228
> 发布时间: 2020-10-09

## 1. 任务

设计并制作一个非接触式物体形状和尺寸自动测量装置。装置能测量被测目标的形状、尺寸、测量头中心点与被测目标之间的距离等参数，并用激光束指示出被测目标的中心位置。背景板竖立放置在目标后 5cm 处。

## 2. 要求

（1）选择某规则形状的平面目标，放在被测目标放置区的中心线位置上，按测量键后开始测量，完成测量后，在装置上显示出该目标物体边长（圆形则显示直径）、几何形状和目标与测量头的距离，总用时不超过 2 分钟。（25分）

（2）更换目标板，在摆放区内中心线上放置目标和背景板，显示距离、形状、尺寸（边长），要求测量用时不超过 2 分钟。（25分）

（3）自动寻找目标测量：测量头处于中心线方向（0°），目标摆放在目标放置区内任选位置；按测试键后，装置自动寻找目标，测量并显示距离、形状、尺寸、用激光笔指示几何中心，用时不超过 3 分钟，越短越好。（30分）

（4）立体目标测量：随机抽取篮球、排球、足球中一个，重复（3）测量，判断球类品种、测量与球表面最近距离。用时不超过 2 分钟。（15分）

（5）其他。（5分）

（6）设计报告。（20分）

## 3. 说明

（1）平面目标为圆形、正三角形、正方形三种，直径或边长 30~40cm，颜色为红、绿、蓝纯色。
（2）背景板为边长约 100cm 的正方形，颜色为白色。
（3）目标板和背景板保持 5cm 左右的间隔，两板总是保持平行放置。
（4）测试装置尺寸不超过长 40cm，宽 30cm，高度自定，测量时不允许添加任何辅助光源。
（5）被测目标的中心与测量头基本上处在同一水平线。
（6）测量过程中要求不得输入数据或用步进、电位器调节等方法改变参数；每次测量只许按同一个按键。
（7）平面目标的平面方向按垂直地面放置，平面法线正对着测量头。
