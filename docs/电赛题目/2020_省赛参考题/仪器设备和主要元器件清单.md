# 2020 年 TI 杯省级大学生电子设计竞赛仪器设备和主要元器件清单

> 来源: https://www.nuedc-training.com.cn/index/news/details/new_id/201
> 发布时间: 2020-08-27
> 说明: TI 杯省级联赛参考命题配套清单

## 省赛测试仪器
- 功率分析仪（电参数测试仪）
- 五位半万用表
- 心电信号模拟器
- 温度计（量程50℃，精度1℃）
- 体温表
- 手持式激光测距仪 / 5米卷尺

## 省赛元器件及设备
- 摄像头
- 二维云台
- 功率管（200V以上，5~10A）
- 自耦变压器（100VA及以上）
- 隔离变压器（100VA及以上）
- 四轮电动小车（投影尺寸不大于250×250mm，质量不大于1.5kg）
- 多旋翼飞行器，最大轴距不大于420mm
- MSP430/MSP432 处理器模块（或Launchpad）
- 红外传感器（20℃~50℃）
- 温度传感器 LMT70
- 模拟前端芯片 ADS1292
- 运算放大器
- ADC
- DAC
