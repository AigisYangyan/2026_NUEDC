底盘运动是**三层级联**，调参分三层、各有各的入口和存储方式。这个文件夹收拢全部底盘调参手册。

```
位置环(定长直行剖面) ─┐
                      ├─→ 速度环(轮速 PID) ─→ 电机
循迹环(循迹外环 PID) ─┘
        外环给「目标轮速」        内环给「PWM」
```

## 三层对照

| 层 | 调什么 | 怎么调 | 存哪 | 手册 |
|---|---|---|---|---|
| **内环 · 速度环** | 轮速 PID Kp/Ki/Kd（左右） | VOFA（`DEBUG/SpeedTune`）调 → **手填代码** | ❌ 不持久化，填进 `chassis.c` 重编烧录 | [速度环增益_手填指南](速度环增益_手填指南.md) |
| **外环 · 循迹环** | 循迹外环差速 PID | 按钮（`TUNE` 组） | ✅ flash（SAVE） | [LineFollow_现场上板与调参手册](LineFollow_现场上板与调参手册.md) |
| **外环 · 位置环** | 梯形剖面 cruise/start/accel/decel + 距离 | 按钮（`DRIVE` 组） | ✅ flash（SAVE） | [定长梯形剖面直行_位置环调参手册](定长梯形剖面直行_位置环调参手册.md) |

## 上板顺序（关键）

1. **先调内环速度环**（速度环不出力，外环白搭）：`DEBUG/SpeedTune` + VOFA 调好 → 手填 `chassis.c` `k_pid_cfg` → 重编烧录。
2. **再调外环**：循迹进 `TUNE` 组、位置环进 `DRIVE` 组，按钮调 + `SAVE` 存 flash。
3. 每层手册里都有：参数清单 / 标定顺序 / 对症调参表 / 安全边界。

## 通用按键（外环按钮调参）

| K1 | K2 | K3 | K4 |
|---|---|---|---|
| 上移 / 加 | 下移 / 减 | 进入 / 确认（SAVE 项=存盘） | 返回 / 退出运行条目（退出即刹停） |

> 内环速度环是手填代码（VOFA 调完填 `chassis.c`），不走按钮——见速度环手册。这是当前已知局限，未来可接进按钮持久化框架。
